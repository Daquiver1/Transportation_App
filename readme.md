# Transportation app

This is a java based transportation app to help users find shortest routes from one location to another in Legon.

## Dependencies

This project has 5 dependencies.
1. opencsv
2. commons-beanutils
3. commons-lang3
4. commons-logging
5. commons-text

The dependencies for this project can be found online.
